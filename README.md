# AcousticLens - Headphone Peak & Dip Detector

イヤホンやヘッドホンのピーク（強調される周波数）やディップ（減衰する周波数）を、A/B比較テストを通じて特定し、周波数特性をビジュアル化するWebアプリケーションです。

## 特徴
1. **安全最優先設計**: Web Audio APIで純粋なサイン波を再生するため、フェードイン・フェードアウト処理を行いポップノイズを排除。初期音量を制限し、安全画面から開始します。
2. **対数周波数スケール**: 人間の周波数知覚に合わせて、対数的に分割された周波数でテストを行います。
3. **アダプティブ・リファイン（詳細測定）**: 1回目の測定で検出されたピークやディップの周辺に自動的に新たな測定ポイントを挿入し、位置と強さを絞り込みます。
4. **確度インジケーター**: ピーク・ディップの周波数がどれだけ絞り込まれているか（位置の確信度）と、隣接周波数との音量差（ピークの強さ）を表示します。
5. **マニュアルスイープ機能**: 測定終了後、スライダーで周波数を動かしながら耳で結果を検証できます。
6. **キーボードショートカット**: テンキーでの回答やスペースキーでの自動交互再生など、効率よくテストが行えます。

## 使い方
1. このフォルダ内の `index.html` をダブルクリック、または任意のWebブラウザ（Chrome, Edge, Safari, Firefox等）で開きます。
2. 画面の指示に従い、イヤホンを装着して音量を調整します。
3. 同意チェックボックスを入れて進みます。
4. 測定周波数範囲（例: 1000Hz〜12000Hz）と初期測定ポイント数を指定して「測定を開始する」を押します。
5. A/B比較でどちらが大きく聴こえるかを回答します。
6. 測定が完了するとグラフとピーク・ディップのリストが表示されます。
7. 「詳細測定（リファイン）」を押すと、検出されたポイント周辺をさらに細かく測定できます。
